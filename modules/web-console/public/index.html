项目简介

GPT QQ Bot 是一个把 GPT 风格助手接入 QQ 群聊和私聊的本地项目。它通过 QQ/OneBot 连接 Codex CLI，在本机保存轻量上下文，并提供 WebUI 管理通道开关、群白名单、维护状态和本地测试。
